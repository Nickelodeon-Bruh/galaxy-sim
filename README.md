# Galaxy Simulation

This project is a small experiment that simulates a galaxy in the browser.  
It combines simple HTML, CSS, and JavaScript to create an interactive visual of stars orbiting around a galactic center.  

## Features
- Dynamic favicon that switches between light and dark mode.
- Animated galaxy effect (stars rotating around a core).
- Runs entirely in the browser — no external dependencies.
